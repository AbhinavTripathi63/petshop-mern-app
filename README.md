# 🐾 PetShop MERN Application

A full-stack PetShop e-commerce application built using the MERN stack (MongoDB, Express, React, Node.js).  
Users can browse pet products/services, add items to cart, place orders via WhatsApp or online payment, and track their orders securely.

---

## 🚀 Features

- User Registration & Login (JWT Authentication)
- Browse Pet Products & Services
- Add to Cart functionality
- Order placement via WhatsApp
- Online payments using Razorpay
- Order history for users
- Secure backend APIs

---

## 🛠 Tech Stack

**Frontend**
- React.js
- Axios
- React Router
- Context API

**Backend**
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication

**Payment Integration**
- Razorpay

## ⚙️ Installation

Clone the repository

Install dependencies

npm install

cd ../frontend
npm install


Run the project
npm run dev
